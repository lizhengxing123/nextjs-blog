### 使用

1. 克隆项目
   ```sh
   git clone https://github.com/lizhengxing123/nextjs-blog.git
   ```
2. 下载依赖包
   ```sh
   npm install
   ```
1. 启动项目
   ```sh
   npm run dev
   ```
4. 打开 http://localhost:3000 查看
